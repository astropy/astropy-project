### Reviving the Matrix Bridge

### Project Team
This proposal is primarily to fund Stuart Mumford at [Aperio Software](https://aperiosoftware.com).
This could also fund other individuals working at Aperio Software as required.

### Project Description / Scope of Work

As of the end of October 2025 the free hosting of the matrix <> slack bridge the project has been using for a number of years has come to an end.
This project requests funding to host a replacement.

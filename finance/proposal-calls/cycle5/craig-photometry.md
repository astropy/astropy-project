### Accessible stellar photometry

### Project Team
Matt Craig will lead the project. Additional participants are from two members of the Astronomy/STEM Alliance with Lick Observatory (ASTRAL), collaborators from others in the upper Midwest who work primarily with undergraduates, and members of the AAVSO.

### Project Description / Scope of Work
This project will focus on a package called stellarphot, which is intended to be
an easy-to-use tool for performing aperture astronomy on a set of images. It is
written to be run either by a script or via a Jupyter notebook-based interface.

This proposal will do three things:

1. Work with users at primarily undergraduate institutions to improve the
   usability and accessibility of the stellarphot package.
2. Develop a workshop and tutorials for the package.
3. Add two features to improve performance of the package: i) calibration using
   an ubercal-style approach that minimizes changes in magnitude across several
   images over many nights and ii) add the ability to work with and save postage
   stamps of images rather than full images.
